Kapitel 1 Überblick

1.1 tl;dr

  • Einführung in die Statistik-Software R
  • keine Vorkenntnisse nötig
  • Dauer: 60-90 Min.
  • online (Zoom)
  • Dozent: Sebastian Sauer
  • Vorbereitung: Konto bei RStudio Cloud anlegen

1.2 Was Sie hier lernen und wozu das gut ist

R ist die führende Software für Datenanalyse und insofern Eintrittskarte in das Datenzeitalter. Sie würden gerne R kennenlernen, aber das kostet Sie zu viel Zeit? Dann kommen Sie zum R-Blitzkurs! In 60 Minuten lernen Sie zentrale Schritte der Datenanalyse. OK, vielleicht reicht die Zeit auch nur für einen ersten Eindruck - aber der erste Schritt ist der wichtigste. Aus dem Inhalt: Daten importieren in R, deskriptive Statistiken erstellen, Diagramme erzeugen, Daten und Ergebnisse exportieren, inferenzstatistische Tests durchführen. Sie brauchen keine Software vorab installieren; im Kurs wird Ihnen eine Online-Version von R bereitgestellt (kostenlos). Es ist aber nötig, sich ein Konto dafür vorab anzulegen, bitte lesen Sie die Details in der Kursbeschreibung.

1.3 Lernziele

Nach dem Absolvieren dieses Kurses …

  • können die Teilnehmis einige zentrale Tätigkeiten der Datenanalyse in R durchführen
  • sind die Teilnehmis in der Lage, selber Analysen mit R durchzuführen

1.4 Inhaltsüberblick

In diesem Kurs werden folgende Themen behandelt:

  • Daten importieren und exportieren
  • Deskriptive Statistiken erzeugen
  • Datendiagramme erzeugen
  • Inferenzstatistische Tests durchführen

Alle Inhalte werden mit dem Statistikprogramm R umgesetzt.

1.5 Vorbereitung vor dem Kurs

  • Legen Sie sich ein Konto an bei RStudio Cloud; s. auch die Hinweise dazu hier. In der kostenfreien Lizenz von RStudio Cloud sind 25h Nutzungszeit pro Monat enthalten.
  • Alternativ können Sie sich R installieren wie hier beschrieben (kostenlos).
  • Werfen Sie einen Blick in das RStudio Cloud Projekt dieses Kurses.

1.6 Modulliteratur

Ein Teil der Literatur ist über viele Hochschulbibliotheken als PDF herunterladen; andere Titel sind offen im Internet verfügbar. Oft müssen Sie per VPN angemeldet (in der Bib.) sein für Volltextzugriff, wenn Sie nicht auf dem Campus sind.
